# 项目规范

这个目录用于存放项目级规则、专项规范、操作说明和辅助资料，主要包括：

- 配图思路说明
- 技术图场景与图类型选择规范
- 技术图组件规范
- 技术图版式模板规范
- 技术图颜色语义规范
- 技术图箭头语义规范
- 技术图文字与文案规范
- 技术图审美检查清单
- 技术图图标使用规范
- 技术图规则总索引
- draw.io 导出命令等工具操作说明
- 临时规划文档
- 补充说明材料

这样做的好处是：

- 主文档专注知识内容。
- 项目规则与正式知识内容分离。
- 工具使用说明有统一归档位置。
- 一些临时规划可以独立存放。

注意：

- 从当前项目规则开始，正式 `.drawio` 文件不再存放在 `项目规范` 中。
- 正式 draw.io 产物统一放在对应文档所在目录下的 `资源/文档名/drawio/`。
- 如果未来需要图片，则放在同级的 `资源/文档名/img/`。
- 具体以项目根目录下的 `AGENTS.md` 为准。
